PROBLEM STATEMENT:

This program makes use of the traffic.csv dataset. Here is a typical line from the file:

 

691,6067 North,Kimball Ave,8/15/06,15600,North Bound:7500/South Bound:8100

 


As you will recall from class, each position in the line (separated by a comma) represents a different "data element". Here is a key for your reference:

ID Address Street Sample Date Total # of Passing Vehicles # of Vehicles by Direction
691 6067 North Kimball Ave 8/15/06 15600 North Bound:7500/South Bound:8100



Write a Python algorithm that opens the file and outputs the lowest traffic count. Then, output the name of all streets whose traffic count is within 2500 (inclusive) of the min. Output one street name per line.

 

For example, if the min value is 500, and the roads within 2500 of this value are "Road A", "Road B", "Road C", then your program should output:

500
Road A
Road B
Road C
