PROBLEM STATEMENT:

Write a program that asks the user to type the name of a file (e.g., "file.txt"), and a letter. Your program should then open the file, and print out all of the lines where the first character starts with the letter

For example, if the user types the following filename:

input1.txt
a

and the contents of input1.txt are:

aspiration
classified
federation

the output should be:

aspiration

 

Notes:

  • Each line ends with the '\n' character. Use split() to get each line.
  • Refer to the String Functions slide to see how you can get the first character from a string
