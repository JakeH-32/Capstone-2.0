PROBLEM STATEMENT:

This program makes use of the Titanic dataset. Here is a typical line from the file:

0,3,Mr. Owen Harris Braund,male,22,1,0,7.25


As you will undoubtedly recall from class, each position in the line represents a different "data element". Here is a key for your reference:

  Survived
(0=False; 1=True)		 Class Name Gender Age Siblings/Spouses Aboard
(0=False; 1=True)		 Parents/Children Aboard
(0=False; 1=True)		 Fare
Value 0 3 Mr. Owen Harris Braund male 22 1 0 7.25



Write a Python algorithm that asks the user for the name of the file containing the titanic dataset, and counts how many males/females survived. Your program should print out each answer to a file called "result.txt".

For example, if 400 men, and 700 women survived, your output file should contain:
400
700

 

NOTE: When you print to a file, make sure you put a new line at the end of each answer!
