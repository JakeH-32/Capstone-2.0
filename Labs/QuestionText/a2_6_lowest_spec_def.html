PROBLEM STATEMENT:

You have been provided with a dataset containing information about various types of Pokemon. Here is an example entry from the file:

25, Pikachu, Electric, 35, 55, 40, 50, 50, 90, 1, FALSE

 

Each position in the line represents a different "data element". Here is a key for your reference:

  ID Name Type Hit Points Attack Defense Special Attack Special Defense Speed Generation Legendary
Value 25 Pikachu Electric 35 55 40 50 50 90 1 False

 

Write a program that asks the user for:

  • a Pokemon Type (e.g., "Electric"). 

 

Your program should then analyze the file and print:

  • The value of lowest Special Defense from a Pokemon with the matching type.
  • The name of the Pokemon of the matching type with the lowest value of Special Defense. In the event of a tie, print out the name of the FIRST Pokemon that matches.

 

EXAMPLE:
If, for example, the Dark Pokemon with the lowest Special Defense is Poochyena (with Special Defense of 30), your program should print out:

30
Poochyena

 

NOTE: The above answer is purely hypothetical. Your answer will be different.

Also, all of the numeric values are positive integers with values less than 999.
