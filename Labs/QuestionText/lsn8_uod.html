PROBLEM STATEMENT:

You’ve been asked to develop a tool to help identify the uniform of the day, based on temperature and wind speed.

The command center has provided the following table for guidance:

Temperature
Wind Speed <40 40 >40 and <60 60 >60
<15 Parkas A-Jackets A-Jackets A-Jackets Blues
15 ABUs ABUs Parkas Parkas Blues
>15 ABUs ABUs Parkas Parkas A-Jackets

Create a Python program that first gets the temperature (in degrees Fahrenheit) and then the wind speed (in knots) from the user. You may assume that all inputs are non-negative numbers.

It should then output the correct response, as described above.
