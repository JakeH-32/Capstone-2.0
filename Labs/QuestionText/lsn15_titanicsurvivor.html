PROBLEM STATEMENT:

This program makes use of the Titanic dataset. Here is a typical line from the file:

0,3,Mr. Owen Harris Braund,male,22,1,0,7.25


As you will undoubtedly recall from class, each position in the line represents a different "data element". Here is a key for your reference:

  Survived
(0=False; 1=True)		 Class Name Gender Age Siblings/Spouses Aboard
(0=False; 1=True)		 Parents/Children Aboard
(0=False; 1=True)		 Fare
Value 0 3 Mr. Owen Harris Braund male 22 1 0 7.25

 

Write a Python algorithm that asks the user for the name of the file containing the titanic dataset. Your program will output the name of the survivor with the lowest fare. Print that person's name in a separate file (named "result.txt")

For example, if the survivor's name is Mr. Owen Harris Braund, result.txt should contain the following:

Mr. Owen Harris Braund
7.25


NOTES:

  • The values used in the above example are purely hypothetical. Your answers will be different.
  • To thoroughly test your code, we created several subsets of the titanic dataset. Use them when developing/troubleshooting your code.
