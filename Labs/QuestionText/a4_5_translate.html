PROBLEM STATEMENT:

Write a function called get_english_word that returns the English translation for a Spanish word. Your function will take the following two parameters:

  1. The spanish word (e.g., "Hola").
  2. A dictionary containing a list of Spanish / English pairs. 

Here is an example of what the dictionary might look like:

 

Key Value
Hola Hello
Adios Goodbye
Aeronave Aircraft

 

Your function should return the English translation of the Spanish word contained within the dictionary.

 

HINT:

Using parameters and return with functions is in the Lessons 11/12 block on your Python Quick Reference guide.
