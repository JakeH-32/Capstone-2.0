PROBLEM STATEMENT:
Write a Python program that asks the user for the number of assessment scores to input, and then gets that many values. Analyze the data using numpy's statistics functions, and print out the following information (one per line):

  • the average (i.e., mean)
  • the number of scores in the 90th percentile (i.e., the number of scores that are at least as high as the 90th percentile score)
  • the number of scores no more than one standard deviation from the mean
