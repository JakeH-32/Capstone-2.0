PROBLEM STATEMENT:

You have been provided with program that gets a number from the user and stores it in a variable called starting_number.

Write a program that prints all of the numbers between starting_number and 0 (inclusive), counting down by 13 each time.

 

For example, if the user types 45, your program should print:

45
32
19
6

Similarly, if the user types 39, your program should print:

39
26
13
0

You may assume that the user will only enter positive numbers.

 

HINT:

  • This problem can be solved using either a for or while loop.
