PROBLEM STATEMENT:

Write the Python code needed to get three values from the user:

  1. A string representing the name of an Air Force Base (e.g., Travis). Store this value in a variable called base_name
  2. A floating point value representing the area of this base in acres. Store this value in a variable called base_acres
  3. An integer representing the number of buildings on this base. Store this value in a variable called building_quantity

IMPORTANT: Your template file contains a print statement that combines these values to produce an output (see Example). Do not modify this line. Simply add your input statements in the space provided.

 

EXAMPLE:

Travis Air Force Base sits on 6383.5 acres and has 1720 buildings on it.
