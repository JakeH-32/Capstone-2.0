PROBLEM STATEMENT:

You are going to be given values for a dictionary. These will be an historic landmark as the key and the location of that landmark as the value. You will continue to receive input until the user types "DONE".

After the dictionary is populated, get a landmark key as an input from the user and print the location where the landmark is. If the landmark isn't in the dictionary, print "Landmark not found".

 

  Example 1
Input(s)

Pyramids
Egypt
Mt Rushmore
South Dakota
Empire State Building
New York City
DONE
Mt Rushmore
Expected Output South Dakota
