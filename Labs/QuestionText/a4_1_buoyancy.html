PROBLEM STATEMENT:

Write a Python function that calculates the buoyancy force for an object. You will name your function calculate_buoyancy, and it will have the following parameters (in this order):

  • Submerged Volume
  • Density
  • Gravity

Our function will return the buoyancy force, which can be computed using the following equation:

Fb = Vs × D × g where Fb is the buoyancy force, Vs is the submerged volume, D is the density of the fluid the object is submerged in, and g is the force of gravity.

NOTE: You do NOT need to call the function.
