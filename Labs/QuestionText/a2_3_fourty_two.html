PROBLEM STATEMENT:

You have been provided with program that gets a number from the user and stores it in a variable called starting_number.

Write a program that prints all of the numbers between starting_number and 0 (inclusive), counting down by 42 each time.

 

For example, if the user types 228, your program should print:

228
186
144
102
60
18

Similarly, if the user types 106, your program should print:

106
64
22

You may assume that the user will only enter positive numbers.

 

HINT:

  • This problem can be solved using either a for or while loop.
