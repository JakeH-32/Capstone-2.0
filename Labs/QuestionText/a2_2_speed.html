PROBLEM STATEMENT:

You can calculate how fast a vehicle is moving in feet per minute using the following formula:

SPEED = RPM * Cir / 12

 

In this formula:

  • RPM is the rotations per minute of the wheel (integer)
  • Cir is the Circumference of the wheel (floating point number in inches)

 

Write a function called speed_fpm that takes RPM and Cir as parameters (in that order), and returns the speed in feet per minute.

 

NOTES:

  • You do NOT need to call this function. Simply define it.
