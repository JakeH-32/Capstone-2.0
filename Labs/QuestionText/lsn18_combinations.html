PROBLEM STATEMENT:

Write a Python function called get_combinations that takes a list of distinct characters as a parameter, and returns a list containing all of the 2-letter combinations of said characters.

For example, given the following list:
my_list = ['a', 'b']

get_combinations(my_list) should return a list containing the following elements:
['aa', 'ab', 'ba', 'bb']

Notes:

  • The ordering of the elements in the list does not matter, so long as all of the 2-letter combinations are contained within the list
