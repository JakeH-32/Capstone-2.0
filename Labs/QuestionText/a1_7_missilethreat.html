PROBLEM STATEMENT:

According to intelligence reports, the North Korean military uses at least 5 different classes of Ballistic Missiles. The names of these missiles, as well as their maximum theoretical range, are provided below:

Missile Class Maximum Range (inclusive, in Kilometers)
Hwasong-6 500
KN-11 1200
Musudan BM-25 4000
Hwasong-14 8000
Taepodong-2 12000


Write a Python program that asks the user to enter the distance to a potential target (in Kilometers). Then, referencing the table, print out all of the Missile Classes that could potentially reach the target (one per line). If no missiles are in range, output "Not in range".

 

EXAMPLES:

  Example 1 Example 2 Example 3
Input 1800 8000 18000
Output Musudan BM-25
Hwason-14
Taepodong-2

Hwasong-14
Taepodong-2

 Not in range

NOTE: The missiles can be listed in any order, so long as only the ones in range are printed.
