PROBLEM STATEMENT:

The following equations estimate the calories burned when exercising (source (Links to an external site.)):

Men: Calories = ( (Age x 0.2017) + (Weight x 0.09036) + (Heart Rate x 0.6309) — 55.0969 ) x Time / 4.184

Women: Calories = ( (Age x 0.074) — (Weight x 0.05741) + (Heart Rate x 0.4472) — 20.4022 ) x Time / 4.184

Write a program using inputs age (years), weight (pounds), heart rate (beats per minute), and time (minutes), respectively. Output calories burned for men and women.

Output each floating-point value (your program will be marked correct so long as it outputs a value within a predefined range)

EXAMPLE:

If the input is:

49
155
148
60

Then the output is:

891.47
580.94
