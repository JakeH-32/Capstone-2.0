PROBLEM STATEMENT:

Write a Python function called get_common_accidents that accepts the following parameters:

  • the name of a file containing crash data (e.g., crashdata_2015.csv)
  • the name of another file containing crash data (e.g., crashdata_2003.csv)

Your function should return a set that only contains the accident_types that are common to both files


As a reminder, here is a sample entry from the crash data file. You can assume that every file is formatted the exact same way.

2015, 1, 5, 0, 2-Car, No Injury, Following Too Closely, W 3rd

 

Each position in the line represents a different "data element". Here is a key for your reference:

  Year Month Day Hour Accident Type Injury Type Reason Location
Value 2015 1 5 0 2-Car No Injury Following Too Closely W 3rd
