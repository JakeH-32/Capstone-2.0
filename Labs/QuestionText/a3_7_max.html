PROBLEM STATEMENT:

You have been provided with a dataset called powerplants.csv that contains information about various power production facilities across the nation. Here is a typical line from the dataset (each line ends with the newline “\n” character):

 

-121.8601395, 38.01598436, 19, GWFPW2_1_UNIT 1, SteamTurbine

 

Each position in the line (separated by a comma) represents a different “data element,” or “column.” The following table shows how the above line can be interpreted:

  Longitude Latitude Power Output (in MW) Name Power Generation Method
Value -121.8601395 38.01598436 19 GWFPW21UNIT 1 SteamTurbine

 

Write a Python program that asks the user for:

  • A power generation method (e.g., “SteamTurbine”)

 

Your program should then read the file and print on separate lines:

  • The maximum power output for power plants of this particular type, and
  • The names of all power plants that are of this particular type, and within 8 MW (inclusive) of the maximum power output

 

EXAMPLES:

  Example 1
Input

SteamTurbine
Output 1164.0
DIABLO_7_UNIT 1
DIABLO_7_UNIT 2
