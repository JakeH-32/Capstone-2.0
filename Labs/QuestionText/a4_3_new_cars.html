PROBLEM STATEMENT:
You have been provided with a table called new_cars that contains information about various automobiles for sale in the United States. Logically, the table can be represented as follows:

 

Unit Price  Model Name Make
$26,660 Bronco Ford
$36,340 4Runner Toyota
$68,199.99 Q8 Audi
$48,475 Silverado Chevrolet

 

Write code that gets 1) Model Name of the car, 2) Make, and 3) the unit price. Your program should then add a new row to the table containing these values.

 

HINT:

  • When you are creating your row, make sure you are adding the columns in the correct order!
