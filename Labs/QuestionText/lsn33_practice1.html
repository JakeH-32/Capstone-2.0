PROBLEM STATEMENT:

Write a Python program that keeps asking the user for positive integers until the user types -1.

Your program will then output the following:

  • # of even numbers
  • Average of all the even numbers (if no even numbers were input, print 0)
  • # of odd numbers
  • Average of all the odd numbers (if no odd numbers were input, print 0)

You may assume that the user will type AT LEAST one number. You cannot assume, however, that the user will enter both an odd and even number.

 

HINT:
Use the mod function to determine if a number is even or odd

print(5 % 2) will print "1" because there is a remainder of 1 when you divide 5 by 2

print(4 % 2) will print "0" because there is no remainder when you divide 4 by 2
