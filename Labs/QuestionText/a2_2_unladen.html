PROBLEM STATEMENT:

The airspeed velocity of an unladen swallow can be estimated using the following formula:

SPEED = 3 * F * A

 

In this formula:

  • F is the number of wing flaps per minute (integer)
  • A is the distance the wing travels (up and down) per wing flap (floating point number in meters)

 

Write a function called speed_of_unladen_swallow that takes F and A as parameters (in that order), and returns the speed in meters per minute.

 

NOTES:

  • You do NOT need to call this function. Simply define it.
