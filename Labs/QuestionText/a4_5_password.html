PROBLEM STATEMENT:

Write a function called get_password that returns the password for a specified website. Your function will take the following two parameters in this order:

  1. The website (e.g., "Amazon").
  2. A dictionary containing a list of website/password pairs. 

Here is an example of what the dictionary might look like:

 

Key Value
Amazon password1234
Facebook aardvark
Twitter AirPower!

 

Your function should return the password of the website contained within the dictionary.

 

HINT:

Using parameters and return with functions is in the Lessons 11/12 block on your Python Quick Reference guide.
