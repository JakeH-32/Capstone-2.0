PROBLEM STATEMENT:

You have been provided with program that gets a number from the user and stores it in a variable called starting_number.

Write a program that prints all of the numbers between starting_number and 0 (inclusive), counting down by 21 each time.

 

For example, if the user types 85, your program should print:

85
64
43
22
1

Similarly, if the user types 51, your program should print:

51
30
9

You may assume that the user will only enter positive numbers.

 

HINT:

  • This problem can be solved using either a for or while loop.
