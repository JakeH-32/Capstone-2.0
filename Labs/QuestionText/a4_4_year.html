PROBLEM STATEMENT:

You have been provided with a file named "cadets.csv" that contains information about (fake) cadets. Here is a line from the file:

 

Karla Webber, Female, CS-17, 2020

 


Each position in the line represents a different "data element." Here is a key for your reference:

 

  Name Gender Squadron Class Year
Value Karla Webber Female CS-17 2020

 

Write a Python program that opens the file and prints out the Class Year of each cadet. For example, if the first three cadets come from 2020, 2019, 2020, the first three lines of output should be:

2020
2019
2020

NOTES:

  • Your solution should print out the years in the order that they were read from the file.
  • Your solution may print out the same year (e.g., 2020) multiple times.
