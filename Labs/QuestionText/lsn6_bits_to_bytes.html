PROBLEM STATEMENT:

The basic unit of computer memory is a bit. A bit can represent two values (on or off, 0 or 1). To represent more information, bits are grouped together in larger quantities. Beyond a bit, the next logical grouping is a byte, where 1 byte has 8 bits. Write an algorithm that prompts the user for a number of bits and then outputs the equivalent number of bytes.

EXAMPLE:

Example 1 Example 2 Example 3
Input 8 800 7234
Output 1.0 100.0 904.25

Note: Output must match exactly.
