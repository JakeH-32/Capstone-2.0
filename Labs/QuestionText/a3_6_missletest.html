PROBLEM STATEMENT:
You have been asked to write a program that analyzes the results acquired from several missile tests executed last week. Write a program that gets from the user the number missile tests preformed, and gets the result from each test (which you may assume are whole numbers) from the user. Your program must print out:

  • The average result for the missile tests.
  • The count of missile tests that scored above the average.
