PROBLEM STATEMENT:

This program makes use of the Titanic dataset. Here is a typical line from the file:

0,3,Mr. Owen Harris Braund,male,22,1,0,7.25


As you will undoubtedly recall from class, each position in the line represents a different "data element". Here is a key for your reference:

  Survived
(0=False; 1=True)		 Class Name Gender Age Siblings/Spouses Aboard
(0=False; 1=True)		 Parents/Children Aboard
(0=False; 1=True)		 Fare
Value 0 3 Mr. Owen Harris Braund male 22 1 0 7.25

 

Write a Python algorithm that asks the user for the name of the file containing some or all of the Titanic dataset, as well as a passenger class (an integer between the values of 1 and 3). Your program should calculate the highest and lowest fare for that passenger class, and print out the result to a file called "result.txt".

For example, if the highest and lowest fare for 2nd class is $20 and $8, respectively, your output file should contain:

20
8


NOTES:

  • The values used in the above example are purely hypothetical. Your answers will be different.
  • Some passengers in the real dataset (like Jack in the movie Titanic--yes, I've seen it . . . don't judge me) have a fare of $0. To thoroughly test your code, we made a separate dataset (titanic2.csv) that removes these values.
