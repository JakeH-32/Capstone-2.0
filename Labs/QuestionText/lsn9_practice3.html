PROBLEM STATEMENT:
Write a program that asks the user for the price of an item in US dollars / cents. Then calculate and print out the equivalent price of the item in:

    • British Pounds (1.25 USD per Pound)
    • Korean Won (1 Won == 0.00083 USD)
    • Mexican Pesos (1 Peso == 53.81 Won)

Print each currency on a separate line.
