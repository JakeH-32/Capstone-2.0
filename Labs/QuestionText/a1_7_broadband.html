PROBLEM STATEMENT:

Choosing the right broadband speed for your internet connection depends on which online activities you plan on engaging while online. Using the table below, determine which activities are possible with a given broadband speed, given in Kilobits per second (Kbps).

 

Activity Short Name Minimum Broadband Speed
(inclusive, in Kbps)
Social Media Social Media 1000
Multiplayer Gaming Gaming 4000
HD Video Conferencing Video Conferencing 6000
File Downloading File Downloads 10000
Stream Ultra HD 4K Video HD 4K Video 25000


Write a Python program that asks the user to enter the broadband speed in Kbps. Then, referencing the table above, print out all the activities (by Short Name) that can be achieved using that speed (one per line). If the broadband speed is less than 1000 Kbps, output "Upgrade to Broadband".

 

EXAMPLES:

  Example 1 Example 2 Example 3
Input 8500 5000 750
Output

Social Media
Gaming
Video Conferencing

Social Media
Gaming

 Upgrade to Broadband

NOTE: The activities can be listed in any order, so long as only the ones in range are printed.
